# Personal Portfolio Website

A responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my skills, projects, education, and achievements.

## 🚀 Live Demo

https://portfolio-resume-indol.vercel.app


## 📂 GitHub Repository

https://github.com/sohail-044/portfolio-resume

## ✨ Features

* Responsive design
* About Me section
* Skills section
* Projects showcase
* Education and Achievements
* Contact section
* Download Resume button
* Dark/Light mode toggle
